How this works:


Elo score is determined via a recency-biased weighted average of a team's scores, with weekly weights determined by the sigmoid function above


Specifically, a given week's weighting for elo score is 1 / (1 + e0.5(week + 6 - weeks played))
Using the same method, weighted standard deviation is also generated, with future weeks' scores randomly generated from the resulting normal distribution
